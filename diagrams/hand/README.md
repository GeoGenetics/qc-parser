# Mermaid ERD diagrams

This archive contains all Mermaid ER diagrams created from the handwritten diagrams in this conversation.

## Contents

- `all_diagrams.md` — all diagrams in one Markdown file
- `diagrams/*.md` — one Markdown file per diagram, including a title and Mermaid code block
- `diagrams/*.mmd` — raw Mermaid source for each diagram

Some relationships were inferred from the handwritten cardinality marks. The notes in the Markdown files describe those interpretations.
